# Use the conditionals to show the conditions

Everything has a cause or a trigger. Therefore, we need the conditionals to help us express such relationship between events. We use the conditionals to show that an event depends on another event.

Example sentence:

If you freeze water, it turns into ice.
The cause/trigger The result of the cause/trigger
The condition The result of the condition

# Quick Check

What is the condition in the sentence - "I would travel around the world if I won the lottery."?

I won the lottery.
